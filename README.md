# Cat Vasya Game

Самостоятельная работа №5 по теме ООП (Python).

Проект представляет собой простую RPG-систему с персонажами и заклинаниями.

---

## Описание

В игре реализованы:

- базовые характеристики персонажа
- разные игровые классы
- расчёт здоровья, урона и защиты
- система маны
- система заклинаний

---

## Структура проекта

- unit.py — абстрактный класс Unit  
- character.py — класс Character  
- spell.py — абстрактный класс Spell  
- spells.py — конкретные заклинания  
- main.py — пример использования  

---

## Unit

Абстрактный класс.

Содержит:
- характеристики персонажа
- список заклинаний
- ману

Методы:
- calculate_max_health()
- calculate_damage()
- calculate_defense()
- add_spell()
- cast_spell()

Создать экземпляр Unit нельзя.

---

## Character

Наследуется от Unit.

Поддерживаемые классы:
- warrior
- mage
- hunter

В зависимости от класса меняются формулы:
- урона
- защиты
- маны

При создании персонажа автоматически рассчитываются:
- максимальное здоровье
- урон
- защита
- максимальная мана
- текущая мана

Все значения округляются вниз.

---

## Spell

Абстрактный класс.

Содержит:
- название
- урон
- стоимость маны

Метод:
- cast()

Создать экземпляр Spell нельзя.

---

## Реализованные заклинания

- Fireball
- IceLance
- LightningBolt

При использовании заклинания:
- проверяется наличие маны
- мана уменьшается
- возвращается урон
